# Enterprise Ecommerce Monorepo

MERN ecommerce project with backend API and frontend client.

## Quick Start

## Prerequisites
- Node.js 18+
- MongoDB (local or managed)

## Setup
```bash
npm ci
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
npm run dev
```

## Quality Commands
```bash
npm run lint
npm test
npm run build
```
